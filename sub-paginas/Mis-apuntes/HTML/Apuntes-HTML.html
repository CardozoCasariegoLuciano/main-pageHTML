Etiquetas:

HTML se maneja con etiquetas, estas se indican con <..> (Mayor/Menor), cada etiqueta que abro la tengo que cerrar con </..>, estas a su vez pueden tener atributos, que se indican luego del nombre de la etiqueta

ejemplo:
   <p>Un Parrafo </p> es una etiqueta normal (etiqueta p)

ejemplo:
   < a hfer="##"> Un Link </a> es una etiqueta con el atributo href="" (etiqueta a)

Estructura principal:

<!DOCTYPE html>
    <html>
        <head>

        </head>
        <body>

        </body>
   </html>

<!DOCTYPE html> Indica que el archivo es un HTML
<html> Todo el contenido de la pagina va a estar entre las etiquetas <html>
<head>Dentro de la etiqueta <head> va el contenido que el usuario no ve, pero que afectan a la pagina
<body> Dentro del <body> va toda la informacion que el usuario ve en de la pagina web

Formularios

<form action="" method=""> ... </form> Estructura principal. Esos 2 atrubutos no son obligatorios para que anden, pero si para que hagan algo ultil

   atributos:
     action="" <----- la pagina que se encarga de procesar los datos del formulario
     method="" < ----- El metodo por el cual el formulario pasara los datos (GET o POST)

Dentro de esas etiquetas uso <label> ... </label> , Agrupan distintas funciones para el formulario

Y dentro de las label voy a usar <input type="" name=""> Que lo que hacen es crear un aspecto concreto del formulario. Dentro de type="" va el tipo de dato de ese campo, en name="" va el nobre que lo identifica

algunos valores para el atributo Type pueden ser:

Text ----- Texto normal
Password ---- Asteristos de contraseñas
checkbox --- un cuadro marcalbe
radio --- igual que el anterior, pero si tienen el mismo valor en el atributo name="", solo podras elejir uno a la vez
button --- es un boton normal
submit --- es un boton que manda la informacion del formulario al sitio elegido en "acction"
color --- te permite elegir un color
range --- te permite elejir un rango

Otros atributos para los Imput:

value="" ---El contenido por defento que tendra escrito ese campo
placeholder="" --- lo mismo pero en un texto de fondo
maxlength="" --- la cantidad maxima de caracteres que permite ese campo (es un numero)


dentro de los label tambien puedo usar: <select> ... </select> y dentro de estos uso <option> ...</option> para crea una lista desplegable donde cada <option></option> es un nuevo elemento

dentro de los label tambien puedo usar: <textarea> ... </textarea>: es un espacio para escribir mas amplio.

<header> ... </header> --- se usa para el contenido que se encuentra al principio de la pagina

<span> ... </span> --- Se usa para contenido generico

<section> ... </section> --- aqui ira en contenido principal

<aside> ... </aside> --- contenido lateral

CSS dentro del HTML

Lo idoneo es tener otro archivo solo para el CSS y conectarlo con una etiqueta en el Head. Pero hay formas de hacerlo dentro de nuestro archivo HTML:

<style type="text/CSS"> ... </style> dentro de la etiqueta Head.
Y dentro de las etiquetas style va todo el codigo CSS (Ver apuntes CSS)

style="... ; ... ;" Esto de usa dentro de una etiqueta como si fuese un atributo

Ejemplo:
<h4 style="color: black; font-size: 15px;> Hola mundo</h4>
Aqui modifico el color y el tamaño de un titulo h4 espesifico
